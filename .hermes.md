# Schoolweb — Website Profil Sekolah

## Stack
- Next.js 16.2.9 (App Router), React 19.2, TypeScript 5 strict
- Tailwind CSS v4, shadcn/ui (base-nova, neutral), Framer Motion 12
- Font: Outfit (heading) + Inter (body)
- Icons: Lucide React, Lightbox: yet-another-react-lightbox

## Commands
```bash
pnpm dev           # Dev server (Turbopack)
pnpm build         # Production build (standalone output)
pnpm start         # Production server
pnpm lint          # ESLint
```

## Architecture
- Semua data masih **static/hardcoded** di `src/data/*.ts` (belum database)
- Target: PostgreSQL + Drizzle ORM + Better Auth (lihat DASHBOARD_PLAN.md)
- Image config: `placehold.co`, `images.unsplash.com`, `img.youtube.com`
- Docker: multi-stage build, standalone output, port 3000

## Design System
- Primary: biru navy `hsl(210,80%,40%)` — header, CTA, link
- Secondary: hijau `hsl(160,60%,40%)` — badge, ikon
- Accent: emas `hsl(35,90%,55%)` — highlight, prestasi
- Neutral 50: `#F8FAFC` (bg), 800: `#1E293B` (text), 900: `#0F172A` (heading)
- Typography: clamp() fluid, Outfit 800/700/600, Inter 400

## Project Rules
- Pendaftaran hanya redirect ke link eksternal — tidak ada form PPDB
- Graphify: kalau ada `graphify-out/graph.json`, query dulu sebelum grep
- Jangan rename `proxy.ts` ke `middleware.ts` (Next.js 16+ convention)
- Data sekolah (nama, alamat, kontak) HARUS dari `settings` table — jangan hardcode
